Capabilities

Unrivaled Capability

Every advanced RAG technique embedded directly into a local-first architecture.

Graph Intelligence

RAPTOR hierarchies, GraphRAG (local / global / hybrid) with community detection, and Code Graph with file→symbol→caller link traversal via code_graph_bridge.

Ingest Everything

54 file formats — PDFs, Notebooks, DOCX, PPTX, code — with SHA-256 deduplication. Smart chunking adapts to content type automatically.

Air-Gapped Privacy

Zero telemetry, zero egress. local_assets_only mode runs a model preflight audit at startup — aborts if any active model would reach the network.

Advanced Retrieval

Hybrid BM25 + dense search, BGE reranker, HyDE, and sentence-window context. Auto query router (heuristic / llm) picks the right strategy — zero tuning.

AxonStore & Gov

HMAC-signed, instantly revocable read-only share keys. Federated queries via @store. Governance audit trail with graceful normal→draining→offline state machine.

MCP & VS Code

31 MCP tools for Claude Code, Codex, Gemini, Cursor & Copilot. 36 LM tools in the VS Code extension. Interactive 3D graph panel with source-line deep-links.

03 / 04 — Copilot Chat

@axon in Copilot

Bring Axon's deep context straight to GitHub Copilot Chat. By invoking @axon, Copilot can search your local indexed RAG database, summarising complex project logic instantly without leaving the editor.

04 / 04 — Project Architecture

Project Isolation & AxonStore

Maintain strict boundaries between knowledge bases. Axon supports up to 5 levels of nested projects — searching a parent fans out to all descendants. Share read-only access via cryptographically signed, instantly revocable HMAC keys, or federate queries across all mounted projects via @store.

  • → Hierarchical projects — parent search fans out to all children
  • → HMAC-signed read-only shares with instant revocation
  • → Graceful: normal → draining → readonly → offline
  • → Full governance audit trail
